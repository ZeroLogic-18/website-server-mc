Buku Panduan Player 📖

Pelajari cara bertahan hidup dan command dari PDF resmi kami.

BAB I: Panduan Memulai Bermain

1. Fase Pertama: Lobi / Spawn

  • Keamanan Akun: Ketik /register [pass] [pass] atau /login [pass].
  • Pahami Aturan: Ketik /rules untuk terhindar dari sanksi.
  • Modal Awal: Ketik /kit wanderer untuk perkakas dasar.
  • Sistem Voting: Ketik /vote dapat kunci gacha gratis.
  • Gacha: Buka peti di area Spawn menggunakan kunci hasil vote.

2. Fase Kedua: Pulau OneBlock Pribadi

  • Buat Pulau: Ketik /ob atau /ob home.
  • Hancurkan Blok: Hancurkan blok tengah terus-menerus. Awas Peti Bonus dan Mob Agresif!
  • Cek Fase: Ketik /ob phase untuk melihat sisa blok.
  • Pusat Kontrol: Ketik /ob menu untuk GUI.
  • Bermain Tim: Ketik /ob invite [nama] untuk mengundang, dan /ob team accept untuk menerima undangan.

3. Fase Ketiga: Ekonomi, Tambang Public, & Klaim Lahan

  • Mencari Uang via Shop: Ketik /shop untuk beli/jual material.
  • Pasar Lelang Player: Ketik /ah sell [harga] untuk jualan, /ah untuk beli.
  • Pertambangan Publik: Ketik /warp tambang.
  • Klaim Lahan Proteksi: Pakai Sekop Emas klik kiri/kanan di 2 sudut, atau berdiri di area lalu ketik /claim. Gunakan /rtp cari tempat luas.
  • Izin Teman: Ketik /trust [nama] izin bangun di klaim-an.

BAB II: Hirarki Rank Member & Hak Akses Command

1. Wanderer (Gratis)

  • /spawn : Teleport kembali ke lobi utama server.
  • /ob home : Teleport langsung ke pulau OneBlock.
  • /rtp : Teleport acak ke koordinat dunia luar yang aman.
  • /sethome [nama] : Menandai lokasi rumah pribadi.
  • /home [nama] : Teleport kembali ke lokasi sethome.
  • /tpa [nama] : Izin berkunjung ke posisi player lain.
  • /balance & /baltop : Cek saldo & peringkat dompet.
  • /pay [nama] [jml] : Transfer nominal uang tunai.
  • /claim : Mengunci kapling tanah proteksi.

2. Gladiator (Donatur v1)

  • Mewarisi seluruh kemampuan Rank Wanderer.
  • /anvil : Membuka menu paron ganti nama item portabel.
  • /workbench : Membuka meja rakit crafting virtual.
  • /feed : Mengisi penuh bar kelaparan instan.
  • /hat : Menjadikan jenis material blok jadi topi.
  • /sit & /lay : Animasi duduk dan rebahan santai.

3. Conqueror (Donatur v2)

  • Mewarisi seluruh kemampuan Rank Gladiator.
  • /enderchest : Membuka brankas tas dimensi portabel.
  • /heal : Pemulihan total darah instan hingga penuh.
  • /repair : Memperbaiki durabilitas alat armor/senjata rusak.
  • /nick [nama] : Mengubah nama panggilan samaran unik chat.
  • /crawl : Animasi merangkak tiarap lewat 1 blok.

4. Overlord (Donatur v3)

  • Mewarisi seluruh kemampuan Rank Conqueror.
  • /fly : Mode terbang bebas anti-gravitasi di langit server.
  • /nick (Warna Hex) : Gunakan gradasi warna warna modern chat.
  • /gmenu main : Katalog barang kosmetik hiburan premium (Pets, Suits).
  • /kit overlord : Mengklaim jatah paket kasta tertinggi.
