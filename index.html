📖 BUKU PANDUAN LENGKAP CAFAALIA SMP

(ULTIMATE GUIDE)

Selamat datang di Cafaalia SMP! Dokumen ini adalah panduan resmi yang memuat segala hal yang perlu kamu ketahui tentang server ini. Mulai dari cara bergabung, bertahan hidup, mencari uang, hingga menguasai fitur-fitur VIP tingkat Dewa.

🟢 BAB 1: SISTEM AKUN & LOGIN (CROSSPLAY)

Server ini mendukung fitur Crossplay, sehingga pemain dari Minecraft Java (PC) dan Minecraft Bedrock (HP/Console) bisa bermain bersama dalam satu dunia.

  1. Registrasi Akun Baru (Wajib)

    Saat pertama kali masuk, akunmu belum aman. Kamu harus membuat password agar barang dan rumahmu tidak dibajak orang lain dengan nama yang sama.

    • Buka chat dan ketik: /register <password_baru> <ulangi_password>
    • Contoh: /register cafaalia123 cafaalia123
  2. Sesi Auto-Login (Login Pintar)

    Kamu TIDAK PERLU mengetik /login setiap kali masuk ke server. Server kita mencatat IP Address-mu.

    • Kamu hanya perlu mengetik /login <password> JIKA kamu ganti HP/PC, ganti jaringan Wi-Fi, atau menggunakan kuota data seluler yang IP-nya berubah-ubah.

🟢 BAB 2: NAVIGASI & TELEPORTASI

Dunia Minecraft itu luas. Jangan sampai kamu tersesat atau repot jalan kaki ribuan blok!

  1. Menu Utama Server: Ketik /menu atau /utama untuk membuka tampilan antarmuka (GUI) yang memuat semua fitur server secara praktis.
  2. Teleportasi Acak (RTP): Berada di Spawn dan bingung mau ke mana? Ketik /rtp untuk diteleportasikan secara acak ke alam liar. Ini adalah titik awalmu membangun base!
  3. Sistem Rumah (Sethome): Menemukan spot bagus untuk rumah? Wajib simpan lokasinya!
    • Ketik /sethome rumahku untuk menyimpan lokasi.
    • Ketik /home rumahku untuk kembali ke rumahmu dari mana saja.
    • (Kapasitas sethome bergantung pada Rank kamu).
  4. Teleport ke Teman (TPA):
    • Ketik /tpa <nama_teman> untuk meminta teleport ke lokasi temanmu.
    • Temanmu harus mengetik /tpaccept untuk menerima, atau /tpacancel untuk menolak.
  5. Kembali ke Pusat Kota: Ketik /spawn kapan saja untuk kembali ke titik awal server.

🟢 BAB 3: BERTAHAN HIDUP (SURVIVAL SYSTEM)

Server ini memiliki beberapa modifikasi ringan agar Survival lebih menyenangkan dan tidak membuang waktu.

  1. Modal Awal (Kit): Jangan mulai dengan tangan kosong! Ketik /kit wanderer untuk mendapatkan pedang, beliung, kapak, dan makanan gratis. Bisa diklaim setiap 24 jam.
  2. Tebang Pohon Instan (Timber): Gunakan kapak jenis apa saja, hancurkan 1 blok kayu paling bawah dari sebuah pohon, dan seluruh batangnya akan otomatis hancur sekaligus.
  3. Tidur Praktis (SinglePlayerSleep): Malam hari terlalu banyak monster? Cukup 1 orang saja yang tidur di kasur, dan cuaca akan langsung berubah menjadi pagi untuk seluruh server!

🟢 BAB 4: ANTI-MALING & KEPEMILIKAN TANAH (CLAIM SYSTEM)

Kita menggunakan sistem GriefPrevention. Selama tanahmu di-claim, tidak ada player lain yang bisa menghancurkan blok, membuka peti, atau membunuh hewan ternakmu.

  1. Cara Mengklaim Tanah:
    • Dapatkan Golden Shovel (Sekop Emas).
    • Klik kanan di tanah pada sudut pertama area rumahmu.
    • Berjalanlah ke sudut berlawanan (secara diagonal) dan klik kanan lagi.
    • Muncul blok emas sementara? Selamat! Area itu sekarang 100% milikmu.
  2. Cara Menambah Luas Tanah: Semakin lama kamu bermain online di server, Claim Blocks (jatah luas tanahmu) akan otomatis bertambah!
  3. Bermain dengan Teman (Trust): Ingin temanmu bisa membuka pintu dan peti di rumahmu?
    • Berdirilah di dalam area rumahmu, lalu ketik /trust <nama_teman>.
    • Untuk mencabut izin, ketik /untrust <nama_teman>.

🟢 BAB 5: EKONOMI & PEKERJAAN (JOBS & SKILLS)

Jadilah orang terkaya di server! Segala aktivitasmu bisa diubah menjadi uang (Koin).

  1. Sistem Pekerjaan (JobSystem):
    • Ketik /jobs untuk melihat daftar pekerjaan (Miner, Farmer, Hunter, dll).
    • Bergabunglah dengan pekerjaan tersebut. Setiap kali kamu menambang atau bertani, uang otomatis masuk ke saldomu.
  2. Sistem Level RPG (AuraSkills):
    • Ketik /skills. Karaktermu akan naik level seiring berjalannya waktu.
    • Semakin tinggi level skill-mu, darahmu (HP) akan bertambah, seranganmu lebih kuat, dan hasil tambangmu bisa berlipat ganda!
  3. Jual Beli (Toko Server):
    • Ketik /shop untuk membuka GUI Toko. Kamu bisa menjual item hasil farming ke server untuk mendapat uang, atau membeli blok untuk membangun rumah.
  4. Perbankan:
    • Cek uangmu: /bal
    • Lihat orang terkaya: /baltop
    • Transfer uang ke teman: /pay <nama_teman> <jumlah_uang> (Contoh: /pay Udin 10000).

🟢 BAB 6: FITUR SOSIAL & HIBURAN

  1. Animasi Karakter (GSit):
    • Ketik /sit untuk duduk santai di tanah.
    • Ketik /lay untuk rebahan.
    • Bisa juga klik kanan langsung pada Stairs/Slab (Tangga) untuk duduk tanpa command!
  2. Toko Kunci (Gacha / CrazyCrates): Pergi ke area Gacha di Spawn. Gunakan kunci yang kamu dapatkan untuk mengundi hadiah (Common, Epic, God Keys).
  3. Dukung Server (Vote): Ketik /vote dan dukung server kita di website. Kamu akan mendapat hadiah dan Kunci Gacha gratis!
  4. Voice Chat: Jika kamu bermain di Java (PC) dan memasang mod Simple Voice Chat, kamu bisa mengobrol langsung menggunakan Mic dengan pemain di sekitarmu (Proximity Chat).
