Buku Panduan Player

Pelajari cara bertahan hidup dan command dari PDF resmi kami.

UNDUH PDF

BAB I: Panduan Memulai Bermain

1. Fase Pertama: Mendarat di Lobi

  • Keamanan Akun: Ketik /register [pass] [pass] atau /login [pass].
  • Pahami Aturan: Ketik /rules agar terhindar dari sanksi.
  • Modal Awal: Ketik /kit wanderer untuk senjata dan perkakas dasar.
  • Voting: Ketik /vote untuk dapat kunci gacha gratis.
  • Gacha: Buka peti gila di area Spawn menggunakan kunci.

2. Fase Kedua: Memulai Pulau OneBlock

  • Buat Pulau: Ketik /ob atau /ob home.
  • Hancurkan Blok: Hancurkan blok tengah terus-menerus. Awas Peti Bonus dan Mob Agresif!
  • Cek Fase: Ketik /ob phase untuk melihat sisa blok.
  • GUI Pulau: Ketik /ob menu mengatur pulau.
  • Co-op: Ketik /ob invite [nama] untuk mabar.

3. Fase Ketiga: Ekonomi, Tambang, & Klaim

  • Toko Server: Ketik /shop untuk beli/jual material.
  • Lelang Player: Ketik /ah sell [harga] untuk jualan, /ah untuk beli.
  • Tambang Publik: Ketik /warp tambang.
  • Klaim Lahan: Pakai /rtp cari tempat, lalu /claim untuk proteksi tanah.

BAB II: Hirarki Rank & Command

1. Wanderer (Gratis)

  • /spawn : Ke lobi utama.
  • /ob home : Ke pulau OneBlock.
  • /rtp : Teleportasi liar acak.
  • /sethome [nama] : Menandai lokasi rumah.
  • /home [nama] : Kembali ke rumah.
  • /tpa [nama] : Izin teleport ke teman.
  • /balance & /baltop : Cek saldo & peringkat kaya.
  • /pay [nama] [jml] : Transfer uang.
  • /claim : Proteksi tanah.

2. Gladiator

  • Mewarisi hak Wanderer.
  • /anvil : Anvil portabel.
  • /workbench : Meja rakit portabel.
  • /feed : Kenyang instan.
  • /hat : Pakai blok jadi topi.
  • /sit & /lay : Duduk dan rebahan santai.

3. Conqueror

  • Mewarisi hak Gladiator.
  • /enderchest : Tas brankas portabel.
  • /heal : Pemulihan darah instan penuh.
  • /repair : Perbaiki alat rusak.
  • /nick [nama] : Ubah nama samaran chat.
  • /crawl : Tiarap lewati 1 blok.

4. Overlord

  • Mewarisi hak Conqueror.
  • /fly : Terbang bebas anti gravitasi.
  • /nick (Warna Hex) : Teks nama warna-warni.
  • /gmenu main : Full kosmetik premium tanpa batas.
  • /kit overlord : Paket senjata legendaris.
